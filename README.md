**Duration:** 8 Weeks
**Focus:** Introduction to AI and its applications in software engineering, ethics, and emerging technologies.

2. **AI Concepts** – Machine learning, deep learning, natural language processing, computer vision, and AI problem-solving approaches.
3. **Popular AI Tools & Frameworks** – Hands-on experience with tools like TensorFlow, PyTorch, Scikit-learn, OpenAI APIs, and Jupyter Notebooks.
4. **AI in Software Engineering** – AI-driven automation, code generation, DevOps integration, and AI-powered software testing.
5. **Future Directives in AI** – Exploring trends, AI governance, and future applications in industry.
