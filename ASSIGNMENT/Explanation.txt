Code Link: # 3. Train/Test Split & # 4. Model Training

Explanation:

Split data into training (80%) and testing (20%) sets to evaluate generalization.

Handling mixed data types.

Robustness to overfitting.

Interpretability via feature importance.

Hyperparameters tuned:

n_estimators (number of trees).

max_depth (controls overfitting).

4. Model Evaluation

Code Link: # 5. Evaluation

Explanation:

Metrics: Confusion matrix, precision, and recall.

Helps measure model effectiveness and ensures high-risk patients are flagged appropriately.
